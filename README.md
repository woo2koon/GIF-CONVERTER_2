# 🎬 GIF Converter v1.7 (Cross-Platform)

Windows와 macOS 모두를 지원하며, 고화질 영상과 유튜브 영상을 손쉽게 고품질 GIF, WebP, MP4로 변환해주는 전문가급 움짤 제작 도구입니다. 각 OS별 하드웨어 가속을 활용한 쾌적한 편집 환경을 제공합니다.

---

## ✨ 주요 기능

### 1. 강력한 미디어 지원
*   **로컬 파일**: ProRes 422 HQ, MOV, MP4 등 다양한 전문가용 코덱 지원.
*   **유튜브 통합**: 유튜브 URL만으로 영상을 즉시 불러와 편집 및 변환 가능.
*   **라이브러리 관리**: 파일별 용량(MB/GB), 포맷, 프록시 상태를 한눈에 확인.

### 2. 고성능 크로스 플랫폼 워크플로우 (Proxy)
*   **OS별 하드웨어 가속**: macOS(`videotoolbox`)와 Windows(`NVENC`, `D3D11VA`)의 하드웨어 가속을 모두 지원하여, 고해상도 영상을 편집용 저용량 파일로 순식간에 생성합니다.
*   **무손실 프리뷰**: 원본의 손상 없이 쾌적하게 구간을 잡고 효과를 확인하세요.

### 3. 정밀한 구간 편집 및 타임라인
*   **프레임 단위 제어**: 정밀한 플레이헤드 이동으로 원하는 찰나를 포착합니다.
*   **인/아웃 트리밍**: 시작점(In)과 끝점(Out)을 자유롭게 설정하고 해당 구간을 무한 반복 재생하여 확인합니다.
*   **키보드 단축키**: 스페이스바(재생/정지), 화살표 키(프레임 이동) 등 편집 효율을 극대화하는 단축키를 지원합니다.

### 4. 동적 키프레임 크롭 (Dynamic Crop)
*   **자유로운 비율**: 16:9, 4:3, 1:1 등 프리셋 비율과 자유형(Free) 크롭 지원.
*   **키프레임 애니메이션**: 시간에 따라 크롭 영역이 부드럽게 이동하는 '동적 크롭' 기능을 통해 피사체를 추적하는 듯한 움짤 제작 가능.

### 5. 고품질 내보내기 설정
*   **다양한 포맷**: GIF, WebP 및 MP4 포맷 지원.
*   **세부 옵션 제어**: FPS(최대 60), 해상도 비율, 디더링(Dither), 색상 수(최대 256), 재생 속도 조절.
*   **오디오 포함**: 필요에 따라 원본 영상의 소리를 움짤에 포함하거나 제거할 수 있습니다.

### 6. 스마트한 설정 및 관리
*   **경로 설정**: 변환된 파일이 저장될 폴더를 자유롭게 지정.
*   **캐시 관리**: 생성된 프록시 파일과 유튜브 다운로드 파일을 간편하게 정리하여 용량 확보.

## 🛠 기술 스택
*   **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+), TailwindCSS
*   **Backend**: Python, Eel (Electron-like GUI)
*   **Engine**: FFmpeg (macOS 및 Windows 하드웨어 가속 활용)

---

## 🚀 시작하기
1. Python 3.9 이상 설치
2. FFmpeg 설치 (Mac의 경우 `brew install ffmpeg`)
3. 필요한 라이브러리 설치: `pip install eel gevent-websocket yt-dlp`
4. `python main.py` 실행
